Portfolio Project Thesis
Title: Personal Portfolio Website - Data Analyst & Business Intelligence Professional

Overview:
A modern, responsive React-based portfolio website showcasing professional expertise in Data Analysis, Business Intelligence, and Applied AI. The portfolio serves as a comprehensive digital representation of professional skills, experience, and projects with an emphasis on modern web design and user experience.

Key Objectives:

Create a professional online presence highlighting 5+ years of experience in analytics, automation, and stakeholder-driven reporting
Showcase technical skills across AI/LLMs, data engineering, BI visualization, and enterprise platforms
Demonstrate real-world project implementations including Agentic AI, telecom KPIs, ETL pipelines, and BI automation
Provide an interactive, modern platform for prospective employers and collaborators
Integrate professional resume and direct contact channels
Core Features:

Portfolio Overview Section: Floating navigation cards providing quick access to key portfolio sections
About Section: Multi-paragraph professional summary highlighting 5+ years of experience and core strengths
Skills Section: Organized into 4 categories - AI/LLMs, Data Engineering, BI/Visualization, Platforms/Tools
Experience Section: Detailed professional experience at Ericsson with 7+ responsibilities
Projects Section: Comprehensive project showcase including the featured Agentic AI project with technical details and outcomes
Education & Certifications: Academic background (BTech from AKGEC) and 5+ professional certifications categorized by domain
Contact Section: Interactive floating contact boxes with direct links to email, phone, LinkedIn, GitHub, and portfolio
Responsive Design: Mobile-first approach with Tailwind CSS
Interactive Elements: Hover animations, smooth transitions, and engaging UI components
Technical Stack:

Frontend: React with functional components
Styling: Tailwind CSS with custom gradients and animations
Deployment: GitHub Pages
Build Tool: npm with React Scripts
Design Philosophy:
Modern, professional aesthetic with dark theme featuring cyan, purple, and green neon accents. Emphasis on visual hierarchy, whitespace, and interactive elements to create an engaging user experience while maintaining professionalism.
